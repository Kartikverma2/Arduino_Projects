# Arduino LED Pattern Generator

This project cycles through multiple LED patterns on button press using an Arduino WiFi R4, 4 LEDs, and a push-button. See `LED_Pattern_Generator_With_Button_Control.ino` for details.

## Components
- Arduino WiFi R4
- 4x LEDs, 4 220ohm resistors, breadboard, tactile switch
